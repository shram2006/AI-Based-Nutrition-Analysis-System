# AI Based Nutrition Analysis System

## Overview
This project is an AI-powered nutrition analysis web application that helps users evaluate the health impact of food consumption using BMI calculations and nutritional data visualization.

## Features
- BMI calculation
- Nutrition analysis
- AI-based diet recommendation
- Interactive charts for calories, protein, fat, and sugar

## Tech Stack
Python
Streamlit
Machine Learning
Pandas
Matplotlib

## Installation
pip install -r requirements.txt

## Run
streamlit run app.py
